How do I install this as a Nagios Plugin

This is for the 3HK video check.
  1. Copy all these files to the libexec directy of Nagios.
  2. Configure Nagios Command to be 'ruby nov_check_video_sites.rb 3hk_video_sites.conf' (FIXME - Do we need to specify absolute paths ?)

File Descriptions

FilenameDescription
httpclient.rb and httpclient directoryBase class for the AcaClient implements HTTP protocol.
aca_client.rbContains classes for AcaClient, AcaClientFactory, etc. : everything needed to create a specialized HTTP client that knows how to talk to ACAs.
aca_client_factory.confContains configuration for several 'prototype' (i.e. preconfigured) ACA clients like 3hk.proxy.novarra.com, vfuk.p2.novarra.co.uk, etc.
nagios_plugin.rbbase class NagiosPlugin for all nagios plugins
nov_check_video_sites.rbNagios plugin to check video sites (make sure that rtsp link is available for videos)
nov_test_ruby_for_nagios.rbA test nagios plugin to make sure that ruby is configured properly on the platform hosting nagios. It returns a WARNING 100% of the time.
test.rbUnit Test Suite for this software. To Execute it simply run 'ruby test.rb'
3hk_video_sites.confConfiguration file for nov_check_video_sites.rb plugin that tests only 3HK.
testregex.rbA tool for testing regular expressions that are configured in the configuration files like 3hk_video_sites.conf
video_check.rbThe workhorse class VideoCheck behind the nov_check_video_sites.rb nagios plugin
